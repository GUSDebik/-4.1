## git commit

**git commit *[файл]*** - перенести в репозиторий изменения всех индексированных файлов

Чтобы перенести изменения файлов внесённых в индекс в репозиторий, использовать команду

```bash+
git commit .
```

Чтобы приложить сообщение к коммиту, использовать параметр `-m`. Пример:

```bash+
git commit -m "init" .
```
